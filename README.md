# file_line_counter

指定したディレクトリ以下のファイル行数の合計を計測する簡易ツールです。

## 前提
- Go がインストールされていること（1.16+ を想定）

## 使い方
```
go run main.go <対象のディレクトリのパス>
```
指定したディレクトリ以下にあるすべてのファイルの行数を合計して表示します。

### オプション
- `-codeonly`  
  ソースコードファイルのみをカウント対象にします。（拡張子判定などでフィルタリング）

## 例
- カレントディレクトリ以下をすべてカウント:
```
go run main.go .
```

- 指定ディレクトリをソースコードのみでカウント:
```
go run main.go -codeonly /path/to/project
```

## 出力例
合計行数を標準出力に表示します（フォーマットは実装に依存）。

## 備考
- サブディレクトリ内のファイルも再帰的にカウントします。
- `-codeonly` の判定ルール（対応する拡張子一覧など）を変更したい場合は、ソースの該当箇所を編集してください。
